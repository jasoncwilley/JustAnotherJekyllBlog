Just Another Jekyll Blog (JAJB)

Another Jekyll Powered Blog That Runs on Liquid

Since every other web developer has a blog these days I decided to create one so I would not feel left out or behind. Just Another Jekyll Blog is powered of course by Jekyll and utilizes both Liquid and Markdown to create reusable templates and blog posts.

Jekyll is very user friendly and anyone who has used simular frameworks will not have any problem learning this one. Not to mention that the syntax is nearly readable. Before you run off to create your own Jekyll Blog take check out mine which is full of useful tutorials I wrote on how I created it
